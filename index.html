Final Porject for Data Visualization -- Mobile Strategy Games

Author: Ziling Wang

Project Description:
As mobile phones plays a significant role in our daily lives nowadays, the mobile games industry is becoming more and more popular.
This project is going to do a deep dive into a dataset with multiple synchronized visualizations.

The topic is mobile strategy games.
And this dataset will be used for this project, which contains 17007 strategy games from the Apple App Store as it claimed.

For this project, we can explore a large number of mobile strategy grams,
find the connections between user ratings and other features, discover how the games developed,
study which feature has the largest influence on user evaluation, and how the price will affect the popularity of the games.

Project Proposal

Alpha Release

Distribution of Average User Rating

It turns out that among all the mobile strategy games,
average ratings with 4.5 took the largest proportion, which is about 1/3;
rating with 4 took the second place;
and the number of mobile strategy games with average rating 3.5 and 5 are very close to each other.

Rating VS Rating Count

From the visualization above,
we can see that most games are Free but has In-app Purchase,
no matter whether its rating is high or low.
We can also find out that very few Totally Free mobile strategy games got rating 5.

Rating VS Count, Price, Size, Name Length

It is quite interesting that
there's a connection between the average user rating and the user rating count.
The average rating keeps increasing as the rating count goes up.
However, rating with 5 is an exception,
probably because as more people rates the game,
it becomes much more difficult to keep the average rating at 5 since that's the highest score we can give.

Another interesting finding is that,
games with a very long name probably will decrease people's ratings.

Size Change for Different Genres

It is quite obvious that
the game size is growing larger and larger.
Let's celebrate the improvement of technology!

And among all the selected genres,
Action seems took the most memory,
probably because action games needs dynamic pictures.
